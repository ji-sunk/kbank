
# Mobile BottomSheet + Left Drawer (Vite + React + TS)
- 햄버거 메뉴 → 레프트 아코디언 드로어
- 딤처리: #000 50% (overlay)
- 하단 바텀시트: 검색조건 설정 (칩/날짜/정렬 및 하단 버튼)

## 실행
```bash
npm install
npm run dev
```

## 파일 위치
- `src/components/LeftDrawer.tsx` : 레프트 메뉴 + 아코디언
- `src/components/SearchFilterSheet.tsx` : 검색조건 바텀시트
- `src/styles/global.css` : 모든 스타일 (모바일 우선)
